# Chuzz Clicker

An over-the-top dual-face clicker inspired by Cookie Clicker. Tap SMALL D CHUZZ and FAT CHUZZ to farm vibes, unleash chaotic power-ups like **ULTIMATE CHUZZ** and **FAT CHUZZ**, and hunt for neon-soaked easter eggs hidden throughout the UI.

## Getting Started

This project is entirely static. To try it out locally run any static web server from the project root, for example:

```bash
python -m http.server 8000
```

Then visit [http://localhost:8000/index.html](http://localhost:8000/index.html) in your browser.
